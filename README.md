# AI/ML Task 2 – House Price Prediction

## Project Overview

This project demonstrates Feature Engineering, Model Optimization, and Performance Comparison using the California Housing Dataset.

## Objectives

- Feature Scaling
- Model Training
- Model Comparison
- Performance Evaluation
- Best Model Selection

## Dataset

California Housing Dataset from Scikit-Learn.

## Models Used

1. Linear Regression
2. Ridge Regression
3. Decision Tree Regressor

## Evaluation Metrics

- RMSE
- R² Score

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Jupyter Notebook

## Results

The models were compared using RMSE and R² Score, and the best-performing model was selected based on evaluation metrics.
